          Examples
        

        

          

            
 Get all users in the database

            
 # GET /users 
 # curl -i -H "Accept: application/json" -H "Content-Type: application/json" localhost:3000/users

            
 Create a new user in the database

            
 # POST /users
 # curl -H "Content-Type: application/@gmail.com","password":"123456", "bio": "example bio"}' http://localhost:3000/users
            

            
Update a specific user with fields (incl only fields in the post, not all).

            
 # PATCH/PUT /users/1
 # curl -H "Accept: application/json" -H "Content-Type: application/json" -X PATCH -d '{"bio": "I'm updating the bio field only."}' http://localhost:3000/users/1
